# goCrawlerHot
go 实现今日热榜

### 运行
```shell
git clone https://github.com/pangxiaobin/goCrawlerHot.git

cd goCrawlerHot

go run main.go
```
### 打包
```shell
cd goCrawlerHot
# 不指定打包名称
go build 
# 指定打包后的名称
go build -o main main.go
```
